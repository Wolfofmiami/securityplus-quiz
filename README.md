# Security+ SY0-701 Quiz PWA

## How to Deploy to Your Phone (Step by Step)

### Step 1 — Push to GitHub
1. Go to github.com and log in (wolfofmiami)
2. Click the + icon → "New repository"
3. Name it: securityplus-quiz
4. Keep it Public, click "Create repository"
5. On your phone/computer, upload ALL these files keeping the folder structure

OR use GitHub Desktop app:
- Open GitHub Desktop → File → New Repository → name it securityplus-quiz
- Copy all these project files into that folder
- Commit and Push

### Step 2 — Deploy on Netlify
1. Go to netlify.com → Log in
2. Click "Add new site" → "Import an existing project"
3. Connect to GitHub → Select securityplus-quiz repo
4. Build settings will auto-detect from netlify.toml
5. Click "Deploy site"
6. Wait ~2 minutes for it to build

### Step 3 — Install on Your Phone
**iPhone (Safari):**
1. Open your Netlify URL in Safari
2. Tap the Share button (box with arrow)
3. Scroll down → tap "Add to Home Screen"
4. Name it "Sec+ Quiz" → tap Add
5. It appears on your home screen like a real app!

**Android (Chrome):**
1. Open your Netlify URL in Chrome
2. Tap the 3-dot menu → "Add to Home screen"
3. Tap Add
4. Done!

### Works Offline
Once installed, the app works 100% offline.
Your XP progress saves automatically between sessions.
